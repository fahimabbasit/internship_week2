# Network Administration Internship - Week 2

## Overview
This repository contains my Week 2 internship assignment, focusing on IP addressing techniques and VLAN implementation. The lab was completed using both Cisco Packet Tracer and GNS3 to strengthen practical networking skills.

## Topics Covered
- FLSM (Fixed Length Subnet Masking)
- VLSM (Variable Length Subnet Masking)
- VLAN Configuration
- Trunk Configuration
- IP Addressing
- Network Connectivity Testing

## Tools Used
- Cisco Packet Tracer
- GNS3
- VPCS
- Windows 11

## Lab Objectives
- Design a network using FLSM and VLSM.
- Configure VLANs on switches.
- Configure trunk links between switches.
- Assign IP addresses to end devices.
- Verify connectivity using Ping.
- Practice subnetting and VLAN implementation.

## Learning Outcomes
During this week, I learned:
- FLSM and VLSM subnetting
- VLAN creation and management
- Trunk port configuration
- Network segmentation
- Connectivity verification and troubleshooting
- Practical implementation using Packet Tracer and GNS3
